# KIM Agent Desktop

## Installation rapide (Windows)

1. Téléchargez le dossier `kim-desktop`
2. Ouvrez un terminal dans ce dossier
3. Exécutez :
```bash
npm install
npm run build-win
```
4. L'installateur sera dans `dist/KIM Agent Setup.exe`

## Installation rapide (Mac)

```bash
npm install
npm run build-mac
```

## Installation rapide (Linux)

```bash
npm install
npm run build-linux
```

## Prérequis

- Node.js 18+
- npm ou yarn

## Développement

Pour lancer en mode développement :
```bash
npm install
npm start
```
